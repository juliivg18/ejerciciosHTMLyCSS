Estilos de sangrado

Reglas para hacer el código más legible, mediante su sangrado, espaciado y colocación de las llaves({}). Los cuatro estilos de sangrado más antiguos del lenguaje de programación C, heredados de otros lenguajes, son:

Estilo K&K

Es el estilo utilizado en el libro sobre programación en C escrito por Kernighan y Ritchie y utilizado en el kernel de Linux( obra de Ritchie y Thomson) 
 if () {
         <body>
                    }

Estilo Allman

Es el estilo utilizado por Eric Allman, creador de muchas utilidades BSD. 
if (cond)
{
         <body>
}
